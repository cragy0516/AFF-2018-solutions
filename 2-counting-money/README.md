## 돈 세기

##### for 250 points

##### pwnned by 4

보통 돈을 셀 때는 천단위로 콤마(,)를 붙인다. 이에 완태는 돈(money)를 입력하면 콤마가 붙여진 금액이 출력되도록 컴퓨터에게 시키고 싶다. 이를 구현하라. (출력 결과에 \n이 없음에 유의하시오)
#hint 스택, 문자열의 끝은 널바이트('\0')

case 1:

입력 :

10000

출력 :

10,000

case 2:

입력 :

1234567

출력 :

1,234,567

case 3:

입력 :

980905

출력 :

980,905
