## 별이 빛나는 밤

##### for 200  points

##### pwnned by 5

숫자를 입력받고, 다음 테스트 케이스들과 같이 별을 찍으시오. 단, o는 알파벳 소문자 'o'를 뜻한다.

case 1 :
입력 :
1

출력 :
*o*

case 2 :
입력 :
2

출력 :
*o**
**o*

case 3 :
입력 :
10

출력 :
*o**********
**o*********
***o********
****o*******
*****o******
******o*****
*******o****
********o***
*********o**
**********o*
